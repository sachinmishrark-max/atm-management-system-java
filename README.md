ATM Management System

A simple ATM Management System made using Java.
This project works on the console and performs basic ATM operations like balance checking, cash withdrawal, deposit, and PIN verification.

Features

- User PIN verification
- Balance inquiry
- Cash withdrawal
- Deposit money
- Simple menu-driven interface
- Basic transaction handling

Technologies Used

- Java
- Object Oriented Programming (OOP)

How to Run

1. Clone the repository
2. Open the project in any Java IDE
3. Compile and run the main Java file

Purpose of the Project

This project was created for practicing Java programming and understanding how ATM systems work logically using conditions, loops, methods, and OOP concepts.

Future Improvements

- Add database connectivity
- GUI interface
- Transaction history
- Multiple user accounts
